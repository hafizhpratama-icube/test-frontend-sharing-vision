# Test Frontend - Sharing Vision 2021

Step by step:

```bash
  git clone git@github.com:hafizhpratama-icube/test-frontend-sharing-vision.git
```
```bash
  composer install
```
```bash
  copy .env.example
  paste dan rename menjadi copy .env

  DB_CONNECTION=mysql
  DB_HOST=127.0.0.1
  DB_PORT=3306
  DB_DATABASE=article
  DB_USERNAME=root
  DB_PASSWORD=28april1999
```

```bash
  php artisan key:generate
```

```bash
  php artisan serve
```
